# Strata Weaver Bridge for Foundry VTT

This is the official companion module for **Strata Weaver**, the ultimate worldbuilding and deterministic simulation tool for Game Masters.

## Features

* **Export Landmarks:** Seamlessly export your Foundry VTT Scene, Map Notes (Pins), and linked Journal Entries into a single, unified JSON package ready to be imported into Strata Weaver.
* **Paste Daily Log:** Bring your Strata Weaver simulations directly into your Foundry VTT chat. Paste the daily weather forecast, tactical overview, and ecosystem events with a single click, perfectly formatted to match the Strata Weaver aesthetic.

## How to Use

1. **Exporting to Strata Weaver:**
   * Open the scene you want to export in Foundry VTT.
   * Open the Chat tab.
   * Click the **Export** button on the Strata Weaver panel.
   * A `.json` file will be downloaded. Import this file into the Strata Weaver web app.

2. **Importing from Strata Weaver:**
   * In the Strata Weaver web app, click **Copy Daily Log** (bottom right panel).
   * Go to Foundry VTT, open the Chat tab.
   * Click the **Daily Log** button on the Strata Weaver panel.
   * The data will be instantly posted to the chat for your players to see.

## Compatibility

* **Foundry VTT:** Version 12+ (Verified for v13)
* **Strata Weaver:** v1.0.0-beta or higher.
